## Hurace.Desktop
* Download
* Make sure Azure CLI is installed (https://docs.microsoft.com/en-us/cli/azure/install-azure-cli)
* Authenticate via `az login`
    * User must be authenticated in the KeyVault
* DB Firewall Rules must be enabled
